3) รายการเส้นเชื่อมของกราฟ
โปรแกรมจะขึ้นว่า:Enter edges (u v w):
ต้องป้อนทีละบรรทัด: u v w
(หมายถึงเส้นเชื่อมจากจุด u → v มีน้ำหนัก w)
(ใส่จำนวนบรรทัดตามค่าของ m)
